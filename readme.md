# Norwegian translation for Authgear

Authgear is a cloud based SaaS platform for identity management. Great service, except for lack of support for norwegian language.

## How?

If you have signed up for the service i assume you have access to the portal.
Follow these steps:

- Select your project
- Select `Languages`
- Scroll down to `Custom Translation` and check `Norwegian`
- Go to `Custom Text` and select `Norwegian` from the dropdown.
- Paste the contents of `translation.json` in the text-field.
- Change the `app.name` and `sms.default.sender` to match your project name
